# CRMFresh

Marketing site for crmfresh.io. Single landing page. Next.js 16 App Router, Tailwind v4, deployed to Vercel.

## Dev

```bash
npm install
npm run dev
```

Visit http://localhost:3000

## Build

```bash
npm run build
```

## Structure

The entire landing page lives in `app/page.tsx`. Root layout and font loading in `app/layout.tsx`. Global styles in `app/globals.css`.
